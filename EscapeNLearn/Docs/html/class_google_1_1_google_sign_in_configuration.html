◆ HidePopups

bool Google.GoogleSignInConfiguration.HidePopups = false

Hides popup UIs from games services.

Used with games signin to show or hide the connecting popup UI and to associate an invisible view for other popups. This is recommended for VR applications. This has no effect if UseGameSignIn is false.
