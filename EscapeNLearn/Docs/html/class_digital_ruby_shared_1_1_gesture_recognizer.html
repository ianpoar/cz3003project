Detailed Description

A gesture recognizer allows handling gestures as well as ensuring that different gestures do not execute at the same time.

Platform specific code is required to create GestureTouch sets and pass them to the appropriate gesture recognizer(s). Creating extension methods on the GestureRecognizer class is a good way.

◆ DisallowSimultaneousExecution()

void DigitalRubyShared.GestureRecognizer.DisallowSimultaneousExecution ( GestureRecognizer  other)

Disallows the simultaneous execution with other gesture.

This unlinks both gestures so this method only needs to be called once on one of the gestures. By default, gesures are not allowed to execute simultaneously, so you only need to call this method if you previously allowed the gestures to execute simultaneously. Pass null to disallow simulatneous execution with all gestures (i.e. you previously called AllowSimultaneousExecution with a null value.

Parameters
otherGesture to no longer execute simultaneously with
