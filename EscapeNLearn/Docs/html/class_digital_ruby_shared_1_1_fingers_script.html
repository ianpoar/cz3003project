◆ CaptureGestureHandler

System.Func<GameObject, bool?> DigitalRubyShared.FingersScript.CaptureGestureHandler

Optional handler to determine whether a game object will pass through or not.

Null handler gets default gesture capture handling. Non-null handler that returns null gets default handling. Non-null handler that returns true captures the gesture. Non-null handler that returns false passes the gesture through.

HashSet<System.Type> DigitalRubyShared.FingersScript.ComponentTypesToDenyPassThrough
get

A set of component types that will stop the gesture from passing through.

By default includes UI components like Button, Dropdown, etc. You can add additional component types if you like, but you should not remove items from this set or clear the set.

◆ ComponentTypesToIgnorePassThrough

HashSet<System.Type> DigitalRubyShared.FingersScript.ComponentTypesToIgnorePassThrough
get

A set of component types that will be ignored for purposes of pass through checking.

By default includes the Text UI component. You can add additional component types if you like, but you should not remove items from this set or clear the set.
